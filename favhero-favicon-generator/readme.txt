== Installation ==

It's easy to get started

1. Upload `ht-ultimate-favicon` to the `/wp-content/plugins/` directory or goto Plugins > Add New and search for `Heroic Favicon Generator`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Select Settings > Heroic Favicon Generator.
4. Upload or select your favicon by clicking Select Favicon.

== Frequently Asked Questions ==

= Q. My favicons are not displaying, or are not correct  =

A. Be sure they are enabled on the settings page and clear all caches (on your server and browser) and hard refresh the page  (CTRL/CMD+SHIFT, and R). If you have a newer version of WordPress ensure that no favicon has been set from Customize > Site Identity > Site Icon.

= Q. I have disabled or deactivated the plugin and want to remove the favicon  =

A. Be sure to delete the favicon.ico from the root of your WordPress install or uncheck `enabled` for all favicons sizes on the settings page before uninstalling the plugin.

= Q. I have a another question or support query =

A. No worries, simply ask on the support section on WordPress.org.
